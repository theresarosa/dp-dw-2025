Como Fiz

Criei o arquivo HTML: Nomeei o arquivo como navegacao.html.
Estrutura da página:
Usei a tag <a> com o atributo href="#id" para criar links internos que direcionam o usuário para partes específicas da mesma página.
Criei três seções principais: Início, Sobre e Contato, cada uma com um id único.
Adicionei um menu de navegação no topo da página com links que levam até cada uma dessas seções.
Para melhorar a aparência, utilizei CSS interno para centralizar o conteúdo e ajustar margens e espaçamentos.

Como Usar

Salve o arquivo navegacao.html em uma pasta.
Abra o arquivo em um navegador e clique nos links do menu para testar a navegação interna entre as seções da página.
