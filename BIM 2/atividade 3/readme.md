Como Fiz

Criei o arquivo HTML: Nomeei o arquivo como tabela-horarios.html.
Estrutura da tabela:
Usei <table> para criar a tabela.
Adicionei o cabeçalho com <thead>, o corpo com <tbody> e o rodapé com <tfoot>.
Utilizei rowspan e colspan para unir células e organizar melhor os horários.
Conteúdo: Montei uma tabela com horários de aulas de segunda a sexta.
Estilo básico: Usei CSS interno para ajustar bordas, cores e centralização do texto.

Como Usar

Salve o arquivo tabela-horarios.html em uma pasta.
Abra o arquivo em um navegador para visualizar a tabela de horários formatada.
