Como Fiz

Criei o arquivo HTML: Nomeei o arquivo como estrutura-basica.html.
Estrutura usada:
<header>: Coloquei o nome do site ("Meu Site") em um <h1>.
<main>: Adicionei um parágrafo simples com texto no corpo da página.
<footer>: Incluí créditos e um contato básico.
Conteúdo: Mantive tudo simples, sem estilos ou navegação, focando apenas nas tags solicitadas.

Como Usar

Salve o arquivo estrutura-basica.html em uma pasta.
Abra o arquivo em qualquer navegador (como Chrome ou Firefox) para visualizar a página.
