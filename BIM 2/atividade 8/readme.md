Como Fiz

Criei o arquivo HTML: Nomeei o arquivo como cadastro.html.
Estrutura do formulário:
Usei a tag <form> para criar o formulário de cadastro.
Adicionei campos de nome, e-mail, senha, cidade e estado.
Utilizei <select> e <option> para criar um menu suspenso com os estados.
Incluí botões de opção com <input type="radio"> e caixas de seleção com <input type="checkbox"> para permitir diferentes tipos de escolha.
Organizei os campos com <label> para identificar cada entrada e deixei o visual mais limpo com um CSS interno simples.

Como Usar

Salve o arquivo cadastro.html em uma pasta.
Abra o arquivo em um navegador para visualizar o formulário completo e testar os diferentes tipos de campos interativos.
