Como Fiz


Criei o arquivo HTML: Nomeei o arquivo como minha-tabela.html.
Estrutura da tabela:
Usei <table> para criar a tabela.
Adicionei uma linha de cabeçalhos com <th> (3 colunas: Nome, Preço, Quantidade).
Adicionei 4 linhas de dados com <td> (totalizando 5 linhas).
Conteúdo: Coloquei dados fictícios de produtos para exemplificar.
Estilo básico: Adicionei border="1" na <table> para mostrar bordas no navegador.

Como Usar

Salve o arquivo minha-tabela.html em uma pasta.
Abra o arquivo em um navegador para ver a tabela organizada.
