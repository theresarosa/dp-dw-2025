Como Fiz

Criei os arquivos form-estilizado.html e form.css.
Estrutura do formulário:
No HTML, montei um formulário simples com campos de texto, e-mail, senha e um botão de envio.
Liguei o arquivo CSS ao HTML usando a tag <link> dentro do <head>.
No form.css, apliquei estilos para deixar o formulário mais bonito e organizado.
Usei seletores de foco e interação, como :hover e :focus, para destacar os campos quando o usuário passa o mouse ou clica neles.
Ajustei cores, bordas, espaçamentos e fontes para melhorar a legibilidade e dar um visual mais profissional.

Como Usar

Salve os arquivos form-estilizado.html e form.css na mesma pasta.
Abra o arquivo form-estilizado.html em um navegador para visualizar o formulário com os estilos aplicados e testar os efeitos de foco e hover.
