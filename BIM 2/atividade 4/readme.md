Como Fiz

Criei o arquivo HTML: Nomeei o arquivo como form-contato.html.
Estrutura do formulário:
Usei a tag <form> para criar o formulário.
Adicionei campos de entrada usando <input> para nome e e-mail, e uma <textarea> para a mensagem.
Incluí um <button> do tipo "submit" para enviar os dados.
Organizei os elementos com <label> para identificar cada campo.
Estilo básico: Usei CSS interno para ajustar espaçamento, bordas e centralizar o formulário na página.

Como Usar

Salve o arquivo form-contato.html em uma pasta.
Abra o arquivo em um navegador para visualizar o formulário de contato e testar o envio das informações.
